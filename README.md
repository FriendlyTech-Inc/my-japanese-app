# 日本語学習アプリ (Japanese Learning App)

モダンなUIと対話型の学習体験を提供する日本語学習アプリケーションです。React NativeとExpoを使用して開発されています。

## 主な機能

- 🌏 言語切り替え（日本語/英語）
- 📚 体系的なレッスン構成
- 🎯 インタラクティブなクイズ
- 🎤 音声認識による発音練習
- ✨ モダンなマテリアルデザインUI
- 📊 学習進捗の追跡

## 技術スタック

- React Native
- Expo
- React Navigation
- React Native Paper
- TypeScript
- i18n (国際化対応)

## 開発環境のセットアップ

1. リポジトリのクローン:
```bash
git clone https://github.com/FriendlyTech-Inc/my-japanese-app.git
cd my-japanese-app
```

2. 依存関係のインストール:
```bash
npm install
```

3. アプリの起動:
```bash
npm start
```
